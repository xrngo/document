# План тестирования проекта «Магазин для сада»

## 1. Цели тестирования
- Проверить корректность работы основных страниц приложения: Главная, Каталог, Товар, Корзина.
- Проверить работу Redux-хранилища: загрузка данных, изменение корзины, ошибки.
- Проверить корректность работы форм (React Hook Form).
- Проверить взаимодействие с локальным backend (JSON-server).
- Оценить удобство пользовательского интерфейса.

---

## 2. Типы тестирования
- **Функциональное тестирование**
- **UI/UX тестирование**
- **Пользовательское тестирование**
- **Smoke-тестирование**

---

## 3. Среда тестирования
- ОС: Windows 11  
- Browser: Yandex browser 
- Backend: JSON-server (localhost:4000)  
- Frontend: Vite + React 18 (localhost:5173)

---

## 4. Инструменты тестирования

## 5. Область тестирования
- Навигация между страницами
- Загрузка списка товаров
- Отображение карточек категорий
- Корзина: добавление, удаление, пересчёты
- Валидация форм
- Корректность API-запросов

---

## 6. Критерии входа
- Приложение успешно запускается
- Backend работает
- Данные доступны

---

## 7. Критерии выхода
- Все критические баги устранены
- Основные пользовательские сценарии работают корректно
