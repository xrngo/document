# Ошибки и предложения по улучшению

## 1. Ошибки

### 1. Удаление товара из корзины работает некорректно  
- Поведение: товар удаляется только со второй попытки  
- Приоритет: высокий  
- Статус: открыт  

### 2. Адаптивная верстка ломается при ширине < 400px  
- Элементы наезжают друг на друга  
- Приоритет: средний  
- Статус: открыт
